# Cloud Fullstack Skills

A Claude Code plugin marketplace for cloud deployment and full-stack development.

## Plugins

| Plugin | Description |
|--------|-------------|
| [modal-deployment](plugins/modal-deployment) | Serverless Python with GPUs using Modal |
| [tanstack-start](plugins/tanstack-start) | Full-stack React with TanStack Start |

## Installation

Add this marketplace to Claude Code:

```bash
/plugin marketplace add ferdousbhai/cloud-fullstack-skills
```

Then install plugins:

```bash
/plugin install modal-deployment@cloud-fullstack-skills
/plugin install tanstack-start@cloud-fullstack-skills
```
