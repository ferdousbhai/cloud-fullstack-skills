# TanStack Start Plugin

Full-stack React framework with SSR, server functions, and Vite bundling.

## When to Use

- Building full-stack React applications
- Server-side rendering with TanStack Router
- Creating server functions with `createServerFn`
- Deploying to Cloudflare Workers
- Integrating TanStack Query with SSR

## Features

- File-based routing with TanStack Router
- Type-safe server functions
- SSR and streaming support
- Cloudflare Workers deployment
- TanStack Query integration

## Installation

```bash
/plugin install tanstack-start@cloud-fullstack-skills
```
